# OP Tracker

開盤溢價率計算與短線強弱觀察工具。

## 功能

- 計算開盤溢價率
- 輸入股票代碼自動帶入股票名稱與市場別
- 查詢股票時嘗試自動抓昨日收盤價與今日開盤價
- 判斷開盤強弱
- 負數直接撤離
- 3%～5% 主要觀察
- 5%～7% 謹慎觀察
- 超過 7% 不追高
- 判斷 9:05、9:10 是否站穩開盤價
- 儲存歷史紀錄
- 匯出 CSV

## 股票資料

第一版使用本地資料：

```text
data/stocks.json
```

欄位包含 `code`、`name`、`market`。

## 自動報價

按「查詢股票」後會先嘗試上市 `tse_股票代碼.tw`，再嘗試上櫃 `otc_股票代碼.tw`。若資料來源、交易時間、休市日或 CORS 導致查詢失敗，仍可手動輸入價格。

## 公式

```text
((今日開盤價 - 昨日收盤價) / 昨日收盤價) * 100
```

## 操作口訣

```text
負數撤離，低於3不看，3到5主觀察，5到7小心追，超過7不追高。
```
