# OP Tracker SPEC.md

## 一、專案目標

OP Tracker 是一個可直接用瀏覽器開啟的開盤溢價率計算與短線強弱觀察工具。使用者輸入股票代碼後，可按「查詢股票」自動抓取股票名稱、昨日收盤價與今日開盤價，並自動計算開盤溢價率與強弱判斷。

## 二、股票查詢

建立函式：

```javascript
async function fetchStockQuote(stockCode) {
  // 嘗試抓上市 tse
  // 如果抓不到，再嘗試抓上櫃 otc
}
```

查詢順序：

1. 先嘗試 `tse_股票代碼.tw`
2. 如果查不到，再嘗試 `otc_股票代碼.tw`
3. 若都查不到，顯示「查無資料，請手動輸入昨日收盤價與今日開盤價」
4. 若 CORS 或連線失敗，顯示「自動查詢失敗，請手動輸入」

查詢結果欄位：

| API 欄位 | 畫面欄位 |
|---|---|
| `n` | 股票名稱 |
| `y` | 昨日收盤價 |
| `o` | 今日開盤價 |
| `ex` | 市場別 |

市場別轉換：

| API 值 | 顯示 |
|---|---|
| `tse` | 上市 |
| `otc` | 上櫃 |

若 `y` 或 `o` 為 `-`、空白或無效數字，不自動計算，保留手動輸入。

## 三、輸入欄位

1. 股票代碼
2. 股票名稱
3. 市場別
4. 日期
5. 昨日收盤價
6. 今日開盤價
7. 9:05價格
8. 9:10價格
9. 備註

## 四、計算公式

```text
((今日開盤價 - 昨日收盤價) / 昨日收盤價) * 100
```

結果顯示為百分比，保留小數點後 2 位。

## 五、強弱判斷

| 開盤溢價率 | 強弱判斷 | 操作建議 |
|---|---|---|
| < 0% | 負溢價，直接撤離 | 直接撤離 |
| = 0% | 平盤開出 | 不觀察 |
| > 0% 且 < 3% | 小幅開高 | 觀望 |
| 3% ~ 5% | 強勢開盤 | 主要觀察 |
| 5% ~ 7% | 強勢偏熱 | 謹慎觀察 |
| > 7% | 過熱開盤 | 不追高 |

若有輸入 9:05 與 9:10 價格，操作建議會再整合站穩開盤價判斷。

## 六、站穩開盤價判斷

| 條件 | 站穩判斷 | 站穩操作建議 |
|---|---|---|
| 9:05價格 >= 今日開盤價 且 9:10價格 >= 今日開盤價 | 站穩開盤價 | 可繼續觀察 |
| 9:05價格 < 今日開盤價 且 9:10價格 < 今日開盤價 | 沒站穩開盤價 | 取消觀察，可能開高走低 |
| 9:05價格 < 今日開盤價 且 9:10價格 >= 今日開盤價 | 轉強觀察 | 可觀察，但不要急追 |
| 9:05價格 >= 今日開盤價 且 9:10價格 < 今日開盤價 | 轉弱 | 小心開高走低 |

## 九、匯出 CSV

匯出 CSV 需包含自動抓到的價格資料、股票名稱、市場別、開盤溢價率、強弱判斷與操作建議。

## 十、操作口訣

```text
負數撤離，低於3不看，3到5主觀察，5到7小心追，超過7不追高。
```
